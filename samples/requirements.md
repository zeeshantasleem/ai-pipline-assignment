# Requirements - Simple Todo App

## What to build
A single-page web application that lets a user manage a todo list.

## Features
- Add a new todo item via a text input and a button
- Mark a todo as complete with strikethrough and visual indicator
- Delete a todo item
- Show a count of remaining incomplete items
- Persist todos in localStorage so they survive a page refresh

## Tech
- Plain HTML, CSS, JavaScript
- Single file output preferred
- Must work in Chrome without any build step

## Acceptance criteria
- Add a todo item
- Mark a todo as complete
- Delete a todo item
- Show remaining incomplete count
- Persist todos after refresh
- No console errors
- Usable on mobile screen 375px wide
